# Forseti AI Conversation

AI chatbot, conversation history, and LLM integration module for Forseti.LIFE.

## Features

- AI-powered conversations and chat
- Conversation history and context management
- LLM integration (OpenAI, Claude, etc.)
- LangGraph orchestration support
- Real-time message streaming

## Installation

```bash
composer require forseti-life/forseti-ai-conversation
drush pm:enable ai_conversation
```

## Configuration

Set your LLM provider and API keys through environment variables or Drupal settings.

## LangGraph Integration

This module integrates with the LangGraph orchestrator for coordinating complex multi-turn conversations and agent tasks.
